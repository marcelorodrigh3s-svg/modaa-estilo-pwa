# 👗 Moda & Estilo - PWA

## Sistema de Gestão para Loja de Roupas

### Tecnologias:
- **Frontend:** React + PWA
- **Backend:** Node.js + Express
- **Banco de Dados:** PostgreSQL

### Como executar:

\`\`\`bash
# Backend
cd backend
npm install
npm run dev

# Frontend (outro terminal)
cd frontend
npm install
npm start
\`\`\`

### Deploy:

- Backend: Railway
